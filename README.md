# README - Excel Practical Exam Project

## Project Title  
**E-Commerce Sales Analysis Dashboard Using Microsoft Excel**

---

## Objective  
The objective of this project is to analyze a large e-commerce sales dataset (100,000 records) using Microsoft 
Excel. The project focuses on data cleaning, advanced formulas, pivot table analysis, KPI creation, data 
visualization, and dashboard development to derive meaningful business insights.

---

## Dataset Information  
- **Dataset Type:** E-Commerce Sales Dataset  
- **Total Records:** 100,000  
- **Total Columns:** Not specified  
- **Key Fields:** Not specified  

---

## Sheet-wise Work Performed  

### 1. Import Dataset  
- Imported raw dataset into Excel  
- Verified column names and structure  
- Checked data consistency and completeness  

---

### 2. Data Cleaning  
- Removed duplicate records  
- Checked and handled missing values  
- Converted **Order Date** into proper date format  
- Created **Month** column from Order Date  
- Standardized data types (numerical and date fields)  

---

### 3. Advanced Formulas  
Used Excel formulas for analysis:  
- IF  
- VLOOKUP  
- INDEX + MATCH  

These formulas were used for dynamic calculations and business metrics.

---

### 4. Pivot Tables  
Created pivot tables for analysis:  
- Sales by Region  
- Sales by Product  
- Sales by Category  
- Monthly Sales Trend  
- Quantity Sold Analysis  

---

### 5. Data Visualization  
Charts used:  
- Clustered Bar Chart  
- Column Chart  
- Line Chart  
- Pie Chart  
- KPI Cards  

---

### 6. KPI Development  
Key Performance Indicators:  
- Total Sales  
- Total Orders  
- Total Quantity Sold  
- Average Sales per Order  
- Number of Customers  
- Top Performing Category  

---

### 7. Dashboard  
An interactive dashboard was created using:  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- KPI Cards  

**Features:**  
- Region-wise filtering  
- Category-wise filtering  
- Dynamic chart updates  
- Interactive business insights  

---

## Key Insights  
- Sales performance varies across different regions  
- Certain product categories contribute most of the revenue  
- Monthly trends help identify peak sales periods  
- Dashboard enables fast, data-driven decision-making  

---

## Conclusion  
This project demonstrates effective use of Microsoft Excel for data analysis, visualization, and dashboard 
creation. Using advanced formulas, pivot tables, KPIs, and slicers, meaningful insights were generated from 
a large dataset. The final dashboard provides an interactive view of business performance.

---

## Tools Used  
Microsoft Excel (Pivot Tables, Pivot Charts, Slicers, Advanced Formulas, Dashboard Design)
